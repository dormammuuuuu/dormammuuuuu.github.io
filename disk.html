DISK SCHEDULER

Disk Scheduling Algorithms is a technique that the operating system uses to determine which requests to satisfy first. The purpose of disk scheduling algorithms is to reduce the total seek time.

Our goal is to calculate each algorithm's exact total seek time and generate a vertical line chart for visualization. SSTF and LOOK were implemented using JavaScript and jQuery
