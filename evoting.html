E-VOTING SYSTEM
  • Description
  • Preview
  • Source Code

This program was made to create an E-Voting system for Senior High School in Arellano University Apolinario Mabini Campus. It’s purpose is to promote transparency and provide honest result of the election voting.

This e-voting system is highly secured, reliable, and user-friendly. The proposed software is designed and tested to work on LAN network.

This program was made using Visual Basic 6.
