QUESTBOT
Description Preview Team Code

QuestBot is a game where users can control and manipulate the movements or directions of a robot. The robot can freely move and will encounter questions before turning on the lights. There is a starting point and a finish line. This system will enhance the mind-skill of the user.



GAME SET UP

Copy the "robot_map.dat" then paste it in C:\emu8086\DEVICES

Copy the DIRECTORY folder then paste it in C:\emu8086\MyBuild\here

This game has 1 level. If the user wants to restart the game , he/she must reset the map manually by moving the robot_map.dat in C:/emu8086/DEVICES.
