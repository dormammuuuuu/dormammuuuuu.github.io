PAGE REPLACEMENT

Page Replacement is needed to decide which page needs to be replaced when a new page comes in. Page fault occurs when the current programs attempt to access the memory page for mapping into virtual address space, but it is unable to load into the physical memory.

Our objective is generate a table for visualization of LRU and Optimal as well as the Page Fault/Hit count and ratio. These program were implemented with the use of JavaScript and jQuery
