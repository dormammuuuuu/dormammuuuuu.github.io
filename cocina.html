COCINA DE MANILA
  • Description
  • Preview
  • Source Code

Cocina De Manila is a Food Delivery system that allows user to manage their orders through our restaurant’s food delivery system in an easier way. By providing a platform to anyone who are craving for their favorite food in our menu without the need to go out.

The purpose of our system is to bring good food whenever or wherever they want.
Anyone can sit back and relax -- 'Cause you order, we deliver.

PREVIEW

HOME PAGE

When the program opens, the Log in page will appear. The user can choose whether to sign up or log in.

SIGN UP PAGE

If Sign up is selected, the user must fill in the required information.

LOG IN PAGE

If Sign up is selected, the user must fill in the required information.

LOG IN PAGE

If Log in is selected, the user must fill in the information he/she used upon registration.

MAIN PAGE

On the main page, the user has four options: The Browse food is used to browse the menu and select the items you will like to order, items added to your cart can be viewed in the Cart menu. You can update/change your account information in the User profile, and the Sign out option to log out your account.

FOOD MENU PAGE

Build up your meal by browsing the menu and select the items you like to order. Items seleted will appear on your cart.

Here is an example on how to order a food in our menu.

CART

The cart contains the user's orders. You can remove or place your order here.

If the user pressed 1, he/she can remove an item from the list.

ORDERS

If the user pressed 2, he/she can now pay the orders. The receipt will appear after placing the orders.

RECEIPT
DELIVERY CHARGES

If the user pressed 3, he/she can see the summary of delivery charges to pay depending on the user's location.

USER PROFILE

The user can update his/her account information such as name, address, email, and password.
